Detailed Description

Represents a scene containing all the occluders. Geometry ray-tracing are performed via a simple ray-tracing procedure.

The ray tracing used in this class is naive, no hierarchical data structure or acceleration is provided. Use GeometryEmbreeScene for a performance oriented scene class. The only acceleration provided is a bounding box test.
